## 2024-05-18 - Prevent Android Data Backup via ADB\n**Vulnerability:** The Android manifest allowed `adb backup` which is enabled by default, this creates a data leak vulnerability allowing physical attackers to extract the private applications storage.\n**Learning:** In older Android apps default behavior is not secure for sensitive information. \n**Prevention:** Always set `android:allowBackup="false"` in the `<application>` tag of `AndroidManifest.xml` to disable data extraction via adb.
